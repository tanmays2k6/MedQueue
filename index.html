Min-Heap — Insert (Bubble Up)

1
New patient is appended to the end of the internal array
2
Compare with parent at floor((i−1)/2)
3
If more critical (lower P-number), swap with parent
4
Repeat upward until heap property is restored
5
Most critical patient always sits at index 0 (root)

Min-Heap — Treat (Sink Down)

1
Remove the root (index 0) — that is the next patient
2
Move last array element to index 0
3
Compare with children: left 2i+1, right 2i+2
4
Swap with the more critical child if needed (sink-down)
5
Repeat until the heap order is fully restored

Sorted Array — How It Works

1
New patient is appended to the end of the array
2
Entire array is re-sorted by priority (P1 first)
3
Ties in priority are broken by arrival order (FIFO)
4
To treat: remove first element — always the most critical
